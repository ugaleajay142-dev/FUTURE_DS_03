## Project Overview
This project analyzes the complete marketing funnel using an interactive Power BI dashboard. It tracks how leads move from generation to deal conversion and final wins. The dashboard highlights stage-wise conversion drop-offs, channel performance, lead quality, and key trends to help businesses optimize marketing strategies and improve overall conversion rates.
